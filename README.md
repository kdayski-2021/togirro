## Портал «ТОГИРРО»

Этот репозиторий содержит код веб‑портала «ТОГИРРО» на базе фреймворка CodeIgniter (PHP). Портал предназначен для проведения конкурсов и образовательных мероприятий с возможностью:
- отображения участников и экспертов;
- проведения онлайн‑голосования;
- регистрации пользователей и записи на мероприятия;
- отображения программы мероприятий и новостной ленты.

## Структура проекта

- **`alpha/web/application/controllers`**: контроллеры CodeIgniter (`Welcome`, `Login`, `Lk`, `Ad` и др.), реализующие основную бизнес‑логику портала.
- **`alpha/web/application/libraries/Design_lib.php`**: библиотека оформления и шаблонизации страниц.
- **`alpha/web/application/views`**: представления (шаблоны) публичной части сайта, личного кабинета и системные шаблоны.
- **`alpha/web/assets/templates`**: статические ресурсы (CSS, JS, изображения) для различных тем оформления (`lk`, `teacher2020` и др.).
- **`.git`**: служебная директория Git‑репозитория.

## Технологии

- **Язык**: PHP 7+
- **Фреймворк**: CodeIgniter
- **База данных**: MySQL/MariaDB (используются таблицы `participant`, `expert`, `vote`, `event`, `point`, `point_user`, `program` и др.)
- **Фронтенд**: HTML5, CSS3, JavaScript (jQuery и кастомные скрипты в `assets/templates/**/js`)

## Основные возможности

- **Публичная часть**:
  - список участников конкурса с фотографиями и категориями;
  - страница отдельного участника;
  - список экспертов и их данные;
  - новостная лента/стрим событий;
  - карта и дорожная карта мероприятия.
- **Голосование**:
  - авторизованные пользователи могут распределять ограниченное количество голосов между участниками;
  - отображение текущего количества голосов за участников;
  - защита от превышения максимально допустимого числа голосов для пользователя.
- **Регистрация и личный кабинет**:
  - регистрация пользователей на мероприятие и отдельные сессии/треки;
  - отправка писем с кодовым словом на e‑mail;
  - учёт квот по площадкам и программам.

## Развёртывание (локально)

1. **Клонируйте репозиторий**

```bash
git clone <URL_репозитория> togirro
cd togirro/alpha/web
```

2. **Настройте виртуальный хост веб‑сервера**

Настройте `Apache` или `nginx` так, чтобы корнем сайта была директория `alpha/web` (где находится `index.php` CodeIgniter).

3. **Создайте базу данных и импортируйте структуру**

- создайте пустую базу данных MySQL/MariaDB;
- импортируйте дамп структуры/данных (файл дампа не входит в репозиторий и должен быть предоставлен отдельно).

4. **Настройте подключение к БД и почте**

- отредактируйте конфиги CodeIgniter в `application/config` (как минимум файлы `database.php`, `config.php`, `email.php`);
- пропишите параметры SMTP‑сервера и адрес отправителя в `email.php`.

5. **Проверьте права на запись**

Убедитесь, что веб‑сервер имеет права на запись в директории, используемые для загрузки файлов и кэша (например, `assets/uploads`, `application/cache`, `application/logs`).

## Полезные маршруты

Примеры основных URL (могут отличаться в зависимости от настроек `base_url` и роутинга):

- главная страница: `/welcome` или `/`;
- список участников и экспертов: `/welcome/show`;
- страница участника: `/welcome/participant/{id}`;
- онлайн‑голосование: `/welcome/vote`;
- регистрация: `/welcome/register5` или `/register` (в зависимости от настроек маршрутов);
- административные разделы (через `Grocery_CRUD`): `/welcome/program?v3` и другие, определённые в контроллерах.

## Требования

- PHP 7.2+ (рекомендуется 7.4);
- расширения PHP: `mbstring`, `mysqli`, `json`, `curl`, `openssl`;
- веб‑сервер Apache или nginx;
- MySQL/MariaDB.
